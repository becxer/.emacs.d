# .emacs.d

becxer's .emacs.d defaults

## dependency

### emacs24.5

    sudo apt-get install checkinstall
    mkdir -p ~/src
    cd ~/src
    wget http://ftp.gnu.org/gnu/emacs/emacs-24.5.tar.gz
    tar xf emacs-24.5.tar.gz
    cd emacs-24.5
    ./configure
    make
    sudo checkinstall
